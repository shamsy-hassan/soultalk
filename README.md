# Soultalk

This is a web application that allows users to chat with each other.

## Running the application

1.  Navigate to the `frontend` directory:
    ```bash
    cd soultalk/frontend
    ```
2.  Install the dependencies:
    ```bash
    npm install
    ```
3.  Run both the frontend and backend servers:
    ```bash
    npm run start
    ```
